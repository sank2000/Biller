### &nbsp;&nbsp;&nbsp;&nbsp; It is **MERN Stack** application for online bill payment with payment methods like PayPal and Stripe.

<br />

### To run it locally :

<br />

> git clone https://github.com/sank2000/Biller.git

> cd Biller

> npm i

> cd api && npm i && cd ..

> cd app && npm i && cd ..

> node server.js

> cd app && npm start
